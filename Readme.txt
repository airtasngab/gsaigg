Aplikasi dibuat dengan menggunakan bahasa pemrograman python 3.12.2 dengan menggunakan beberapa modul yang telah dicantumkan dalam dile requirements.txt
Requirements Lainnya:
-Install file Tesseract di "C:\Program Files\Tesseract-OCR\tesseract.exe"
.
.

Penjelasan Singkat:
-Menu WXREV dengan input sandi WXREV dienkripsi menjadi data cuaca dan ditampilkan sebagai konten infografis pada template gambar yang sudah dibuat sebagai backgroundnya.
-Menu Prakgel (prakiraan gelombang) dengan input link 'https://maritim.bmkg.go.id/area/pelayanan/?kode=M.02&hari=1', 'https://maritim.bmkg.go.id/area/pelayanan/?kode=M.01&hari=1', 'https://maritim.bmkg.go.id/area/pelayanan/?kode=M.02&hari=2', dan 'https://maritim.bmkg.go.id/area/pelayanan/?kode=M.01&hari=2' untuk memparsing informasi data prakiraannya dan ditampilkan sebagai konten infografis pada template gambar yang sudah dibuat sebagai backgroundnya.
-Menu Pdce3h (prakiraan dini cuaca ekstrem) dengan input gambar kemudian teks pada gambar akan diekstrak sebagai data dan ditampilkan sebagai konten infografis pada template gambar yang sudah dibuat sebagai backgroundnya.
*Menu pdce3h memerlukan file Tesseract untuk mengekstrak teks dari gambar.
.
.
